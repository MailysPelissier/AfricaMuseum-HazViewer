- fonction download_hazminer qui permet le téléchargement des données hazminer :
	- url = `http://localhost:8080/geoserver/webGIS/ows?service=WFS&version=1.1.0&request=GetFeature&typeName=webGIS:events` + `&outputFormat=application/json`; (lignes 1563 et 1564)
	- url = `http://localhost:8080/geoserver/webGIS/ows?service=WFS&version=1.1.0&request=GetFeature&typeName=webGIS:events` + `&outputFormat=application/json` + `&CQL_FILTER=` + encodeURIComponent(cql_filter); (lignes 1567 et 1568)
- fonction download_co qui permet le téléchargement des données citizen observer (pas utilisée) :
	- url = `http://localhost:8080/geoserver/webGIS/ows?service=WFS&version=1.1.0&request=GetFeature&typeName=webGIS:citizen_observer` + `&outputFormat=application/json`; (lignes 1743 et 1744)
	- url = `http://localhost:8080/geoserver/webGIS/ows?service=WFS&version=1.1.0&request=GetFeature&typeName=webGIS:citizen_observer` + `&outputFormat=application/json` + `&CQL_FILTER=` + encodeURIComponent(cql_filter); (lignes 1747 et 1748)
- fonction create_paragraph_download_text qui crée le texte de téléchargement des paragraphes liés à un évènement (hazminer) :
	- let url = `http://localhost:8080/geoserver/webGIS/ows?service=WFS&version=1.1.0&request=GetFeature&typeName=webGIS:paragraphs` + `&outputFormat=application/json` + `&CQL_FILTER=` + encodeURIComponent(cql_filter); (lignes 1972 et 1973)
- couche de susceptibilité des tremblements de terre : 
	- source: new ol.source.TileWMS({
	url: 'http://localhost:8080/geoserver/webGIS/wms',
	params: {
	    'LAYERS': 'webGIS:landslide_susceptibility',
	    'TILED': true,
	    'VERSION': '1.1.1',
	    'FORMAT': 'image/png',
	    'CRS': 'EPSG:3857'
	},
	crossOrigin: 'anonymous',
	serverType: 'geoserver',
	transition: 0,
	}), (lignes 2316 à 2328)
- couche des rivières :
	- source: new ol.source.TileWMS({
	url: 'http://localhost:8080/geoserver/webGIS/wms',
	params: {
	    'LAYERS': 'webGIS:HydroRIVERS_flow5max',
	    'TILED': true,
	    'VERSION': '1.1.1',
	    'FORMAT': 'image/png',
	    'CRS': 'EPSG:3857'
	},
	crossOrigin: 'anonymous',
	serverType: 'geoserver',
	transition: 0,
	}), (lignes 2335 à 2347)
- couche évènements hazminer (se remplit selon l'emprise de l'écran) :
	- url: function(extent) {
	    return `http://localhost:8080/geoserver/webGIS/ows?service=WFS&version=1.1.0&request=GetFeature&typeName=webGIS:events` + `&outputFormat=application/json` + `&bbox=` + extent.join(',') + `,EPSG:3857`;
	}, (lignes 2403 à 2406)
- couche évènements citizen observer (se remplit selon l'emprise de l'écran) :
	- url: function(extent) {
	    return `http://localhost:8080/geoserver/webGIS/ows?service=WFS&version=1.1.0&request=GetFeature&typeName=webGIS:citizen_observer` + `&outputFormat=application/json` + `&bbox=` + extent.join(',') + `,EPSG:3857`;
	} (lignes 2416 à 2419)
